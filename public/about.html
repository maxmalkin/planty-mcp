MCP server for your plants

> ABOUT

An MCP server for plant care management. Track plants, watering schedules, growth measurements, and images.

> HOW TO USE
1. GENERATE API KEY Visit the home page and click "Generate API Key"
2. SAVE YOUR KEY Copy and save your API key somewhere safe. You won't be able to see it again.
3. CONFIGURE MCP CLIENT Add this server to Claude Desktop or ChatGPT:

Endpoint: https://planty-mcp.onrender.com/sse
API Key: planty_live_xxx...
Auth: Bearer token
4. START MANAGING PLANTS Use the MCP tools to track your plants, water logs, growth, and more!
> FEATURES
• Plant collection management
• Watering tracking and scheduling
• Growth measurement logging
• Plant image references
• Multi-user support with API keys
> AVAILABLE TOOLS
• add_plant - Add new plant
• list_plants - List all plants
• get_plant - Get plant details
• update_plant - Update plant info
• delete_plant - Remove plant
• water_plant - Record watering
• get_watering_history - View water logs
• get_watering_schedule - Plants needing water
• add_growth_log - Log growth measurements
• get_growth_logs - View growth history
• add_plant_image - Add image reference
• get_plant_images - Get plant images
